# PRD — 二刀流開發助手控制台

版本：v1.0（2026-06-04）｜狀態：待用戶確認

## 1. 專案背景

用戶在 Codex 與 Claude Code 兩個 AI CLI 中已安裝 45 個 skills（含 1 個自製 dual-ai-workflow，備份於本 repo），並持續從小紅書等來源發現新 skill 與好用提示詞。

## 2. 專案目標

做一個免安裝、雙擊即開的本機 HTML 控制台，讓新手用戶不用記任何指令，就能：查 skill 用途 → 複製觸發句 → 貼到 CLI 調用；以及快速複製常用提示詞。

## 3. 專案痛點

| # | 痛點 | 對應功能 |
|---|---|---|
| 1 | skill 裝太多，記不住用途，更記不住觸發句 | Skill 目錄＋觸發句一鍵複製 |
| 2 | 常用提示詞散落各處，每次重打 | 提示詞庫一鍵複製 |
| 3 | 網路上看到新 skill 想裝，怕資安問題 | 安檢 SOP＋安檢提示詞＋風險標色 |
| 4 | 不會寫程式，無法自己維護工具 | 維護方式＝把新內容丟給 Claude 即可 |

## 4. 功能清單

1. Skill 目錄：45 個 skill（含 1 個自製 dual-ai-workflow）的中文用途、適用情境、注意事項、1–3 句觸發句（一鍵複製）
2. 提示詞庫：5 大分類約 15 張卡（專案啟動／開發過程／版本控制／整合串接／上線部署＋安全檢查），一鍵複製
3. 全域搜尋：關鍵字即時過濾 skill 與提示詞
4. 風險標示：每個 skill 標 低／中／高 風險（如 baoyu-danger-* 標高）
5. 安檢 SOP 頁籤：安裝前檢查流程、紅旗清單、安檢提示詞
6. 資料維護：data/*.yaml 為唯一資料來源，scripts/build.py 重建 index.html；日常維護可直接請 Claude 代勞

## 5. 不做的事（本期範圍外）

- 不做線上服務、不需帳號登入、不連網
- 不自動安裝/移除 skill（只提供說明與安檢）
- 不做 CLI 內的查詢 skill（方案 B，未來可加）

## 6. 驗收標準

- 雙擊 index.html 可開啟，搜尋「翻譯」能找到 baoyu-translate
- 點任一複製按鈕，貼上內容正確
- danger 系列 skill 顯示高風險紅色標示
- 45 個 skill（含 1 個自製 dual-ai-workflow）與全部提示詞卡片皆有顯示
